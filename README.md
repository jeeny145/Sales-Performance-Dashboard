# Sales Performance Dashboard

## Project Overview
This project analyzes monthly sales performance using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code

## Dashboard Features
- Monthly Sales Trend
- Monthly Profit Trend
- Monthly Orders Analysis
- Profit Margin Analysis

## Key Insights
- Continuous sales growth observed.
- December achieved highest sales and profit.
- Profit margin improved throughout the year.
